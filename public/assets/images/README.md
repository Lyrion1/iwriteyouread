# Favicon Placeholder

The favicon.ico file should be created and placed in this directory.

## Instructions for creating a favicon:

1. Create a 512x512px image with your logo or initials
2. Use a favicon generator like:
   - https://favicon.io/
   - https://realfavicongenerator.net/
   
3. Generate multiple sizes:
   - favicon.ico (16x16, 32x32, 48x48)
   - apple-touch-icon.png (180x180)
   - favicon-16x16.png
   - favicon-32x32.png
   - android-chrome-192x192.png
   - android-chrome-512x512.png

4. Place all files in `/public/assets/images/`
